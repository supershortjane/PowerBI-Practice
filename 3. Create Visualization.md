## PowerBI-Practice

### Introduction<br/> 
There are multiple ways to visualize the data. Each type of visualization has different purpose.<br/>
**Bar Chart** :to see the data among various categories<br/>
**Line Chart** :to see the data over time<br/>
**Pie Chart**, **Tree maps** :to see the proportion of the data<br/>
**Map** :to see the data over space<br/>
**Scatter plot** :to see the relationship between data<br/>
**KPI card** :to visualize the target data<br/>
In this pratice, I build a bar chart with the visualization that PowerBI provided. Also, I get more visuals from AppSource to create a world cloud. 
Then, create a slicer to filter measure data and categorical data.
